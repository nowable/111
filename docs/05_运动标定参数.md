# OriginCar 运动标定参数记录

## 1. 测试环境

- 日期：2026-05-26
- 平台：OriginCar / RDK X5
- 控制话题：`/cmd_vel`
- 标定脚本：
  - `ros2 run origin_competition_auto motion_calibration`
- 底盘服务：
  - `origincar-base.service`
- 测试方式：
  - 车辆落地测试
  - 每次动作结束后自动发布零速度停车

## 2. 直行速度标定

测试命令格式：

```bash
ros2 run origin_competition_auto motion_calibration --step forward --linear <速度> --duration 3.0
```

| `linear.x` | 持续时间 | 实测距离 | 估算平均速度 | 现象 |
|---:|---:|---:|---:|---|
| 0.03 | 3.0 s | 8 cm | 0.027 m/s | 可移动，适合精细靠近 |
| 0.05 | 3.0 s | 13.5 cm | 0.045 m/s | 稳定低速，适合默认巡航 |
| 0.08 | 3.0 s | 24 cm | 0.080 m/s | 较快，适合直线路段 |
| 0.10 | 3.0 s | 32.5 cm | 0.108 m/s | 更快，暂不作为默认速度 |

结论：

- `linear.x` 与实际速度基本成正比。
- `0.03` 适合靠近二维码、障碍物附近微调。
- `0.05` 适合作为第一版自动流程默认巡航速度。
- `0.08` 可以作为较空旷直线路段的快速速度。
- `0.10` 暂时只作为测试上限，不建议比赛初版常用。

## 3. 转向标定

测试命令格式：

```bash
ros2 run origin_competition_auto motion_calibration --step left --linear 0.05 --angular 0.2 --duration 3.0
ros2 run origin_competition_auto motion_calibration --step right --linear 0.05 --angular 0.2 --duration 3.0
```

已测结果：

| 动作 | `linear.x` | `angular.z` | 持续时间 | 现象 |
|---|---:|---:|---:|---|
| 左转 | 0.05 | +0.2 | 3.0 s | 转弯方向正常 |
| 右转 | 0.05 | -0.2 | 3.0 s | 转弯方向正常 |

当前结论：

- 左右转方向映射正常。
- 暂未精确测量转弯半径和角度。
- 第一版自动流程可以先使用时间控制和视觉反馈，不强依赖精确转弯半径。

## 4. 第一版推荐控制参数

```text
AUTO_APPROACH_SPEED = 0.03
AUTO_CRUISE_SPEED   = 0.05
AUTO_FAST_SPEED     = 0.08
AUTO_TURN_LINEAR    = 0.05
AUTO_TURN_ANGULAR   = 0.2
STOP_REPEAT         = 5
CMD_RATE_HZ         = 10
```

用途建议：

- 靠近二维码、障碍物、目标点：`linear.x = 0.03`
- 常规自动巡航：`linear.x = 0.05`
- 空旷直线快速移动：`linear.x = 0.08`
- 常规左转：`linear.x = 0.05, angular.z = +0.2`
- 常规右转：`linear.x = 0.05, angular.z = -0.2`
- 停车：连续发布 `linear.x = 0.0, angular.z = 0.0`

## 5. 后续补充项

后续如果需要更精确控制，再补测：

- `angular.z = 0.3`
- `angular.z = 0.4`
- 固定 `linear.x = 0.05` 时，一圈或半圈所需时间
- `/odom` 中的位姿变化
- 不同地面材质下的速度差异

当前阶段不要求精确转弯半径，优先保证：

- 方向正确
- 速度可控
- 停车可靠
- 自动流程能闭环跑通

## 6. 高速段标定（2026-06-14，拆掉 0.12 上限后）

车落地前进 3s,卷尺量距,速度=距离/3。

| cmd linear.x | 实测距离 | 实测速度 |
|---:|---:|---:|
| 0.15 | 45 cm | 0.15 m/s |
| 0.25 | 75 cm | 0.25 |
| 0.30 | 90 cm | 0.30 |
| 0.40 | 132 cm | 0.44 |
| 0.50 | 154 cm | 0.51 |
| 0.60 | 185 cm | 0.62 |
| 0.70 | 215 cm | 0.72 |
| 0.80 | 247 cm | 0.82 |
| 0.90 | 280 cm | 0.93 |
| 1.00 | 310 cm | 1.03 |

结论:
- **cmd ≈ 实际 m/s(≈1:1,含滑行约 +3%);线性到 1.0 未见饱和 → 底盘 ≥1.03 m/s,真极限更高未测。** 速度不是瓶颈。
- **前轮安装偏:直行略向左漂**。视觉循迹会自动纠(PID 跟线);开环段(桥/odom返航)受影响,建议机械微调或加软件 `steering_trim`。
- `motion_calibration` 与 `auto_mission` 的 `max_linear` 上限已从 0.12 放到 3.0(2026-06-14)。
- 完赛建议:直线巡航 0.3、0.5m 窄环 0.15–0.2(限制来自 10Hz 控制率+窄道,非电机)。
